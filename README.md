# 电商领域搜索分词。

所有的代码采用纯C编写，字典用Redis存储。

Redis 字典的格式如下：

set 户 '{"0":"户外","1":"户型"}'

set 背 '{"0":"背包","1":"背景"}'

就像上面展示的，词条的首汉字作为键值存储在Redis中。

具体的代码处理在main.c有体现，有很详细的注释。
